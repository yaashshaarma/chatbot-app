# chatbot-app

## Overview
This chatbot uses DialoGPT and Streamlit for conversational interaction.

## Setup
1. Install dependencies:
   ```
   pip install transformers torch streamlit
   ```
2. Run the Streamlit app:
   ```
   streamlit run app.py
   ```
